# CLAUDE.md

Guidance for Claude Code when working in this repository.

## Git

- **Do NOT push to any git remote.** Never run `git push` (or any command that
  publishes to a remote) under any circumstances.
- **Do NOT commit — not even locally.** Never run `git commit` (or `git add` in
  preparation for one). Leave all changes in the working tree for the user to
  review and commit themselves.
- Read-only git commands (`git status`, `git diff`, `git log`) are fine.
